Notes taken during translating/playtesting in Arabic:

You're expected to know EasyRPG's translation tool before reading this.

General info

Putting English text in the middle of an Arabic sentence (or word) anywhere flips the said English text example: CSE instead of ESC, elppa instead of apple, so they have to be reversed.

Item/Weapon/Armor/Magic Descriptions & Meta Description

Change the order of words from left-to-right because EasyRPG can't render text from Right-To-Left (except message boxes)

Message boxes (when characters speak)

Type Arabic like you normally would without reversing anything besides English text (!مرحى)

Code language for meta.ini file

I didn't playtest this with others a lot, but I'll assume it's correct because it makes sense (and because I got no complaints). the code language for Arabic translations have to be ar as putting for example ar-IQ means only poeple living in Iraq can play it (or I guess only people with their system local set to Arabic Iraq)
